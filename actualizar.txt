ERROR: Waiting for your frontend dev server to start on http://localhost:1420/
→ Sobreescribiste el vite.config.ts. Repáralo:

## 📦 COMPILAR INSTALADORES CON GITHUB ACTIONS

### ▶️ PASO 1 — Sube los cambios y crea un tag de versión

cd /home/imanowl/Desktop/PROYECTOS/CUADERNO/cuadernoprofesoradoGitHub/tauri-cuaderno

⚠️ El tag vX.X es lo que dispara la compilación.
