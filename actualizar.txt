ERROR: Waiting for your frontend dev server to start on http://localhost:1420/
→ Sobreescribiste el vite.config.ts. Repáralo:

cat > /home/imanowl/Desktop/PROYECTOS/CUADERNO/cuadernoprofesorado/tauri-cuaderno/vite.config.ts << 'EOF'
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
  server: {
    port: 1420,
    strictPort: true,
  },
  build: {
    outDir: 'dist',
  },
})
EOF

Luego ejecuta npm run tauri dev de nuevo.

4. Haz clic en el workflow completado
5. Baja hasta la sección "Artifacts"
6. Descarga:

   - .msi  → Windows
   - .dmg  → macOS
